Acorn, the new code to make designs.

How Acorn works:

The Acorn coding language is based off of a 4 by 4 grid of 16 squares. Pixel1 is the top left square and pixel16 is the bottom right. The numbers increase left to right and go down the rows.

The code starts with "file.acorn { }" and the name of your file goes inside the brackets. You also need to select the program as "drawing".

Acorn version 1.0 will include a HTML file you can access built from the Acorn code that can be embedded in websites because Acorn does not work for that. Acorn is meant to be the easier coding language to learn.

Acorn version 1 automatically generates HTML code for your picture that is made with Acorn. To access the different files in the project, including the HTML file, check the files located on the left.
