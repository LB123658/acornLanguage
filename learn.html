Acorn, the new code to make designs.

Learning Acorn:

Acorn is designed to be very easy to learn.

All you have to do is put the colors of the squares you want in the code. The code has "pixel1" throught "pixel16" in a grid of 16 squares. The top four are 1 through 4, and they increase from there, left to right.

Part of the code:

file.acorn {Example code}
program::drawing
pixel1 == green
pixel2 == blue
pixel3 == orange
pixel4 == purple
end program::

The code above only has four color squares, making a horizontal line that is (left to right) green, blue, orange, and then purple. You can use color names, as well as the nexadecimal codes for these as well.
